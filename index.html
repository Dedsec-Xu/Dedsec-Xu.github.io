DatasetImgLabel

DatasetImgLabeler is a image annotation tool for researchers to prepare datasets in ICDAR2015 format. (similar to tzutalin/LabelImg)

It supported arbitrary quadrilateral annotations, machine learning assisted annotations, and dataset visualization.

It reached a hmean of 83.7% in ICDAR 2015

Rubik’s Cube Robot

Rubik’s Cube Robot is a project that I worked on in Sophomore year. It is capable of solving any rubix cube in 3 seconds. A FPGA board serves as the Lower machine, which controls rotors by generating PWM waves. A cellphone serves as the Upper machine. It calculates the formula and control the FPGA board via Bluetooth. I led a 3-men team to finish this project.

  • Teammate: Shiyin Ge and Zhengzhuo Xu

ROS LiDAR Robot

The robot is developed for the TI cup electronics design competition. It is based on STM32 MCU (lower machine) and raspberry pie (Running ROS). It can navigate to IoT devices and charge them wirelessly. The charge strategy is based on WIFI information exchange and Greedy Algorithm. I led a 5-men team to finish this project in 2 weeks. This project won third place in the TI cup competition.

SmartExchange

During the last summer, I applied for Summer Research Internship at Rice University EIC Lab, where I participated in the SmartExchange research group.

My major work in SmartExchange research group was to rewrite part of the codes from pytorch ML framework with C++, so as to implement the algorithm into chips to evaluate its energy consumption. The C++ implementation of SmartExchange model itself is removed from the repo. But the framework I developed can be potentially used to implement other models.

The work had been submitted with me as a co-author.
